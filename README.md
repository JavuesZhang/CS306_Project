## Usage
1. Place your `sample_taxi.csv` in `resource/`;
2. `cd src`
3. Process data with `data_process.py`;
    ```bash
    python data_process.py
    ```
    Then you will get some data files in `resource/`.
### Static Analysis
```bash
cd ../src
streamlit run streamlitDemo.py
```
### Dynamic Trajectory
```bash
cd ..
npm install
npm start
```
